snaplet-oauth
=============

oauth client snaplet

support oauth providers

  - [weibo](http://weib.com)
  - [google](https://developers.google.com/accounts/docs/OAuth2WebServer)
  - github

Build & Tests
=============
  
  - `make init rebuild`
  - `make test`

Demo
=============

  1. `cd example`
  2. update `src/Keys.hs.default` and rename to `Keys.hs`
  3. make clean prev
